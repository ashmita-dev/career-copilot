# Career Copilot

Career Copilot is an AI-powered career guidance platform that helps aspiring developers identify skill gaps, discover recommended projects, and follow personalized learning roadmaps to achieve their target tech roles.

## Features

* Skill Gap Analysis
* Career Readiness Score
* Missing Skills Detection
* Personalized Learning Roadmaps
* Recommended Portfolio Projects
* Modern Interactive Dashboard
* Responsive User Interface

## Tech Stack

### Frontend

* React
* React Router
* Axios
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js
* PostgreSQL
* Neon Database

## How It Works

1. Select a target role.
2. Choose your current skills.
3. Generate a career analysis report.
4. View:

   * Match Percentage
   * Missing Skills
   * Recommended Projects
   * Learning Roadmap
